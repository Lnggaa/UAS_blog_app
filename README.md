# 📝 Mini Blog Platform

Platform blog sederhana berbasis web untuk menulis, berbagi, dan berinteraksi dengan artikel. Dibangun dengan arsitektur REST API (backend) dan React (frontend).

## 🚀 Tech Stack

### Backend

- **Runtime:** Node.js
- **Framework:** Express.js
- **ORM:** Prisma
- **Database:** MySQL
- **Authentication:** JWT (JSON Web Token)
- **Password Hashing:** bcryptjs

### Frontend

- **Framework:** React.js (Vite)
- **HTTP Client:** Axios
- **Routing:** React Router DOM

## 📦 Cara Menjalankan

### Prasyarat

- Node.js (v16 atau lebih tinggi)
- MySQL (v8 atau lebih tinggi)
- npm atau yarn
